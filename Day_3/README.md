# Day 3 – Multi Agent Lab

This repository contains the Day 3 lab for the Advanced Agentic AI Systems Engineering program.

The project uses LangGraph to build a multi-agent research workflow that collects and analyzes information, evaluates research quality, and generates a final Markdown report.

## Files
- `multiagent_lab.py` – Report Agent workflow
- `Dockerfile` – Docker configuration
- `output_report.md` – Generated report
